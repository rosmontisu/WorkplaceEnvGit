## 오토 작동 방식
1. adb client device 객체 생성
2. 현재 에뮬레이터의 스크린을 캡쳐
   - adb shell screencap 방식
   - win32api 로 핸들 가져오는 방식(후킹)
3. cv/ocr로 스크린에서 탐색 후 좌표 추출
4. 만들어둔 루틴 확인

5. 루틴에 맞는 좌표로 adb를 통해 입력
   - adb touch 
   - android miniTouch
